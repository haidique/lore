# Code standards

Coding conventions governing how Lore source is written.

## What this folder is

Per-language, per-area conventions for error handling, logging, task spawning, testing, comments, and similar concerns. Each page is a Code-Standard doc — imperative rule statements paired with rationale and good/bad code examples.
